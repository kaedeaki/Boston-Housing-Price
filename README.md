# Boston-Housing-Price  

**Name of project**: Boston Housing Price  
**Aim**: To predict which features impacts to the housing price in Boston with machine learning regression.  
**Solution**: Used statistical method. Analyzed which coefficients are significant the price. Analyzed multicollinearity, P-values, and coefficients.  
**Results**: The regression model explains about 76.7% of the variation in the data. Charles River dummy variable is the most significant to increase the price. Nitric Oxide concentration is the most impact feature to decrease the price.
